# final-donkey
This is the updated donkey jump project which was built based on the React.
I have changed donkey failure logic and added donkey life count. For now, I set 3.

## Installation
  ```
  npm install 
  ```
  or if you have already installed yarn
  ```
  yarn install
  ```
## Run the project
  ```
  npm run start
  ```
  or
  ```
  yarn start
  ```
